# Awesome-weather

#### [링크](https://expo.io/@seanny/Awesome-weather)

## 🌵 프로젝트 소개

🌧 Open Weather API와 Expo-location를 활용한 React Native 날씨앱, User location-based weather app using React Native.
Expo 앱으로 위 링크의 QR코드를 인식하면 바로 실행할 수 있습니다.
